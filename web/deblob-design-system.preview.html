Mascot Placement Matrix

Allowed

  • Product identity, empty states, onboarding, docs headers.
  • Unknown, candidate, matched, promoted, and abstain state illustrations.
  • Small construction diagrams where the schema core is visible.

Prohibited

  • Malformed, quarantine, security, audit, and machine output.
  • Promotion controls or decisions that imply the mascot approves.
  • Any state that recolors the mascot body red, amber, blue, or lilac.
